# ❤️ AI-Powered Heart Failure Prediction System

## 📌 Project Overview

This project is a Machine Learning-based web application that predicts the likelihood of heart disease using patient health information. The model is trained on the Heart Failure Prediction dataset and deployed as an interactive Streamlit application.

---

## 🚀 Features

* Interactive Streamlit Web Application
* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Statistical Analysis
* Hypothesis Testing
* Logistic Regression, Random Forest & Gradient Boosting Comparison
* Hyperparameter Tuning using GridSearchCV
* Feature Importance Analysis
* SHAP Explainability
* Heart Disease Prediction with Probability
* Risk Level Classification
* Health Recommendations
* Model Persistence using Joblib
